■ 다른 PC에서 쓰는 방법 (설정 몰라도 됨)

1) 이 PC(개발 PC)에서 prepare-portable.bat 더블클릭
   → SellermatePortable 폴더 생성 (.env + .env.defaults 포함)

2) SellermatePortable 폴더 통째로 복사

3) 새 PC: Node.js + Git 설치

4) 붙여넣기
   · D:\naverrank 가 이미 있고 예전에 깔았던 PC:
     → 폴더 안의 .env.defaults 가 있으면 FIX-ENV.bat 실행
     → 또는 예전 D:\naverrank\.env 파일 삭제 후 통째로 덮어쓰기
   · 새 폴더: 아무 경로에 붙여넣고 START.bat 실행

5) START.bat 더블클릭 (EXE 대신 이것도 됨)

※ EXE 실행 시에도 .env.defaults 가 있으면 Legacy JWT .env 는 자동 교체됩니다.
※ 키는 Git에 올라가지 않습니다.
